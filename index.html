1. PRIVACY STATEMENT & BOUNDARIES

Project Z operates fundamentally as a local-first application workspace environment. Our system data flow logic guarantees that all objective mappings, telemetry logs, parsing variables, roadmap data entries, and generation prompts remain containerized strictly within your native physical sandbox boundary layer.

2. DATA INGEST AND SCOPE EXCLUSION

We do not collect, monetize, analyze, or intercept your operational roadmap content inputs. Inference arrays are computed entirely using on-device silicon processing. When an operator proactively submits an email address into our system waitlist communication node, this string package is encrypted and securely routed via TLS tunnel systems directly to our database middleware partner, Formspree (Formspree, Inc.).
